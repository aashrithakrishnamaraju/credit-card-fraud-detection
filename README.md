# 🏦 Loan Default Prediction

## 📌 Problem Statement
Predict whether a loan applicant will be approved or rejected
based on financial and demographic data.

## 📊 Dataset
- Source: Kaggle - Loan Prediction Problem Dataset
- 614 records | 13 features

## 🛠️ Tools Used
- Python, Pandas, Scikit-learn, Matplotlib, Seaborn
- Google Colab

## 🤖 Model
- Logistic Regression
- Accuracy: ~80%

## 📈 Key Findings
- Credit history is the strongest predictor of loan approval
- Higher income applicants have better approval rates
